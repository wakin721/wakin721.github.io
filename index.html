环境准备与运行 (Setup)

Neri 的架构分为前端 (Flutter Desktop) 和后端 (Python AI/数据处理服务)。请确保您的开发机器已安装 Python 3.10+ 和 Flutter SDK。

1. 安装 Python 依赖

在项目根目录下，使用 pip 安装 requirements.txt 中的包（主要包含 PyTorch, OpenCV 等推理与图像处理库）：

pip install -r requirements.txt content_copy

2. 资源与模型就绪

确保您的 /res 目录下存在所需模型与字体文件：

  • res/model/26m_p2_0224.pt (检测模型)
  • res/species_database.db (本地名录数据库)

3. 启动前端 Flutter 应用

进入 frontend 目录并获取依赖，最后启动桌面端：

cd frontend
flutter pub get
flutter run -d windows content_copy

注：在生产环境中，您可以通过编译 launcher.c 来一键启动前后端进程。
